# 🍽 City Analysis – Data Analysis Internship Task

This project is part of my **Data Analysis Internship** where I performed city-wise analysis on a restaurant dataset using **PostgreSQL**.

## 📊 Task Objective

The goal of this task was to analyze restaurant data based on cities and extract useful insights.

The analysis includes:

* Identifying the **city with the highest number of restaurants**
* Calculating the **average rating of restaurants for each city**
* Determining the **city with the highest average rating**

## 🛠 Tools & Technologies

* PostgreSQL
* SQL
* Restaurant Dataset
* Data Analysis Techniques

## 📈 Key Insights

* **New Delhi** has the highest number of restaurants in the dataset.
* Restaurant ratings vary across cities, indicating differences in customer experience and service quality.

## 💡 Skills Demonstrated

* SQL Queries
* Data Aggregation
* Data Analysis
* Insight Extraction

This task helped strengthen my practical understanding of **SQL-based data analysis and dataset exploration**.
